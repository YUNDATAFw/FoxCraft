# Fox_Craft
这是一款为《我的世界》设计的辅助工具，提供多种功能，帮助玩家更好地体验游戏。

* Fox科技 v4.2.0 更新日志
** 更新内容

1.全面重写页面布局，使用Google Material Design3 组件，重构底层框架。
2.增加 _快捷键、 灵动显示_ 功能。
3.美化布局UI，图标，
4.提升部分功能频率：
		“自动攻击”(30次/s -> 100次/s)
		“自动崩服”(800次/s -> 1000次/s)
5.修改“展框拿取”属性为单击。
6.加强权限判断管理。
7.加入切换当前输入法选项。
8.加入版本管理机制，允许非最新版本运行(从此版本起,低于此版本依然无法运行)。
9.修复更新提示无法正常显示的问题。
10.修复了一些已知bug。
