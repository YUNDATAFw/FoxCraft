# 🦊 FoxPlugin 狐狸插件

＊ FoxPlugin 是基于 ”狐狸科技” 框架运行的 Json格式 插件脚本

## 📚 功能介绍
- **[功能拓展]** 该插件可以让您在 应用 中增加自定义额外的功能与运行逻辑，让功能不局限在内置功能中；
- **[UI自定义]** 可通过内置接口修改部分可自定义布局，文本等控件；

## 📑 快速上手
如果您想开发属于您的插件，试试这份快速上手指南：

1.首先您需要先创建一个.json为后缀的脚本，或者您也可以选择一个插件模板

基础模板： 

2.根据FoxPlugin的Json格式编写程序指令集

3.使用FPC编译器，将您的脚本代码编译成插件包

4.在狐狸科技中导入您的插件包，即可体验您插件中的功能
